一个表单提交到两个servlet

(2011-10-13 23:00:51)
转载▼
标签：

杂谈

 分类： javaweb
一个FORM表单中有两个SUBMIT按钮（例如“修改”“删除”），只有一个ACTION处理页面，在此ACTION页面中如何区分是处理“修改”呢还是处理“删除”？

1、假如FORM表单是这样的：

     <INPUT Type="Submit" Name="Action" Value="修改">

     <INPUT Type="Submit" Name="Action" Value="删除">

    你可以在ASP中读取Request("Action")的值，如果用户选择“修改”，则这个变量是"修改"。

 

2、可以做一个脚本函数，比如：

    <Script Language="JavaScript">

    function modify()

    {

    document.form1.action="modify.jsp";

    document.form1.submit();

    }

    

    function delete()

    {

    document.form1.action="delete.jsp";

    document.form1.submit();

    }

    </Script>

    

    <form name="form1" action="">

    <INPUT Type="Button" Name="Modify" Value="修 改 " onClick="modify()">

     <INPUT Type="Button" Name="Delete" Value="删 除 " onClick="delete()">

    </form>

    这样可以实现将多个按钮发送到不同网页中。
